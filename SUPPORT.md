# Support

## Questions and bugs

Use the repository that matches your work:

| Topic                                      | Repository                                                      |
| ------------------------------------------ | --------------------------------------------------------------- |
| Specification, schema, and format docs     | [twilic/twilic](https://github.com/twilic/twilic)               |
| JavaScript / TypeScript implementation     | [twilic/twilic-js](https://github.com/twilic/twilic-js)         |
| Rust implementation                        | [twilic/twilic-rust](https://github.com/twilic/twilic-rust)     |
| Go implementation                          | [twilic/twilic-go](https://github.com/twilic/twilic-go)         |
| Zig implementation                         | [twilic/twilic-zig](https://github.com/twilic/twilic-zig)       |
| Ruby implementation                        | [twilic/twilic-ruby](https://github.com/twilic/twilic-ruby)     |
| Python implementation                      | [twilic/twilic-python](https://github.com/twilic/twilic-python) |
| Benchmarks                                 | [twilic/benchmark](https://github.com/twilic/benchmark)         |
| Organization policies and shared templates | [twilic/.github](https://github.com/twilic/.github)             |

Open a **GitHub Issue** in the relevant repository with:

- What you are trying to do
- What you expected vs. what happened
- Version or commit, language/runtime, and minimal reproduction steps when applicable

Search existing issues before filing a duplicate.

## Security issues

See [SECURITY.md](./SECURITY.md). Do not use public issues for vulnerability reports.
